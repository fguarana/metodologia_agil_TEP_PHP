# Pesquisa sobre PHP

## Descrição do Projeto

Este repositório foi desenvolvido como parte de uma atividade colaborativa acadêmica com o objetivo de pesquisar a linguagem de programação PHP, utilizando práticas de metodologia ágil para organização das tarefas e acompanhamento do desenvolvimento do projeto.

A pesquisa aborda desde a história do PHP até suas principais características, aplicações no mercado, vantagens, desvantagens e exemplos de código.

# Objetivos

## Objetivo Geral

Pesquisar e apresentar informações relevantes sobre a linguagem de programação PHP.

## Objetivos Específicos

* Estudar a história e evolução do PHP;
* Identificar os criadores da linguagem;
* Apresentar as principais características do PHP;
* Demonstrar exemplos de código;
* Explicar áreas de aplicação da linguagem;
* Analisar vantagens e desvantagens;
* Mostrar empresas que utilizam PHP;
* Pesquisar o framework Laravel;
* Desenvolver organização colaborativa utilizando metodologia ágil.
